Quick Overview

This document is meant to get you up and running as easily as possible. For a more in depth explenation, head over to the Code Explenation page.

These are the steps to adding this tool to your webpage.
  1. Add the 1 CSS file to a folder on your server. (Most common path is root/css/style.css)
  2. Add the 2 JS files to a folder on your server. (Most common path is root/js/files.js)
  3. Copy the "Toggler" html below
  4. Paste it into a single container on page
  5. Update the IMG src's on the .viewer section, and the .thumbnail section
  6. Increment the layer numbers
  7. Copy the "CSS Link" html below
  8. Paste it in the head of your website
  9. Make sure that your link path is correct
  10. Copy the "Scripts" html below
  11. Paste is just before the </body>
  12. Make sure that the src paths are all correct
