1.a Parent Container

The parent container is a single div that holds the viewer and toggle div's. 
<div class="main-layer-toggler-container" id="layer-toggler-1">
    viewer
    toggle
</div>

Above you can see the start <div> and end </div> tags for the parent container.

Note that this has an ID of layer-toggler-1. If you add a second toggler to the same page, you will need to have the second ID set to layer-toggler-2 if 3 on one page, then the third must be 3, and so on.

1.b Viewer

The Viewer is the left half of the tool. It holds images as layers. 
<div class="viewer">
    <img src="./images/sky-graphic/blue-brackground.png" alt="" class="layer layer-1">
    <img src="./images/sky-graphic/skyline.png" alt="" class="layer layer-2">
    <img src="./images/sky-graphic/house.png" alt="" class="layer layer-3">
    <img src="./images/sky-graphic/grass.png" alt="" class="layer layer-4">
</div>

Above, you will see all the viewers code. A few things to point out are

  1. The main div holding the image layers, must have the class name of .viewer on it
  2. Image names do not matter
  3. Eeach image needs to have a class name of .layer & a class name of which layer it is. (I.E. .layer-1, or .layer-2)
  4. If a page has more than 1 toggler on it, you must continue the layer numbers.
    • For example, say you have the first toggler with 4 layers, then the second toggler would not start with .layer-1, it would start with .layer-5.

2) The JavaScript

This is a full representation of all the js used for the "Layer Toggler"

This is really three sections, 1) Initialization 2) checkIfActive and 3) updateparentHeight

$(window).ready(function(){
            
    updateparentHeight();

    $('.checkbox').click(function(){
        var thisLayer = $(this).attr('id');
        checkIfActive(thisLayer);
    })
});

function checkIfActive(thisLayer) {
    
    var thisLayerID = '#'+thisLayer;
    var thisLayerClass = '.'+thisLayer;

    if ($(thisLayerID).is(':checked')) {
        $(thisLayerClass).show(225);
    } else {
        $(thisLayerClass).hide(225);
    }
};

function updateparentHeight() {

    var totalTogglers = $('.main-layer-toggler-container').length;

    for (i = 1; i <= totalTogglers; i++) { var parentID="#layer-toggler-" + i; var thisSelector=parentID +' .viewer>:nth-child(1)';
        var childImageHeight = $(thisSelector).height();

        $(parentID).css('height', childImageHeight)
    };

}

2.a Initialization

$(window).ready(function(){

        updateparentHeight();

        $('.checkbox').click(function(){
            var thisLayer = $(this).attr('id');
            checkIfActive(thisLayer);
        });
});

This is saying,

  • when the window is ready
  • run the function "updateparentHeight()"
  • and listen for an event where an element with the class of ".checked" is clicked on.
  • If the event happens,
  • set the variable "thisLayer" to the ID of the element clicked on
  • then, run the function "checkIfActive" and run it with thisLayer

2.b Check If Active

function checkIfActive(thisLayer) {

    var thisLayerID = '#'+thisLayer;
    var thisLayerClass = '.'+thisLayer;

    if ($(thisLayerID).is(':checked')) {
        $(thisLayerClass).show(225);
    } else {
        $(thisLayerClass).hide(225);
    }
};

This is saying,

  • Set the variable thisLayerID to "#" and thisLayer (thisLayer was passed in from the first argument "thisLayer")
  • Then set the variable thisLayerClass to "#" and thisLayer
  • If thisLayerID is checked on click
  • Then animate in the layer (Note that the state "checked" will update before the JS runs, so if it's checked, and you click on the check box, it will uncheck, then run the JS)
  • If it's not checked
  • Hide the layer image


2.c Update Parent Height

function updateparentHeight() {

    var totalTogglers = $('.main-layer-toggler-container').length;

    for (i = 1; i <= totalTogglers; i++) { 
        var parentID="#layer-toggler-" + i; 
        var thisSelector = parentID +' .viewer>:nth-child(1)';
        var childImageHeight = $(thisSelector).height();

        $(parentID).css('height', childImageHeight)
    };

};

This is saying,

  • Set the variable "totalTogglers" to the amount of togglers there are on this page
  • Then run this section of code X amount of times, where X is the number of "totalTogglers"
  • In this section, set the variable "parentID" to the CSS selector #layer-toggler-i (Where I is the number of times this code ran, starting at 1)
  • Then set the variable "thisSelector" to the child ".viewer"'s last child from the var "parentID" (This is actually just grabbing the height of the image so that when the browser screen size changes the .viewer container will always be the right height)
  • Then set the variable "childImageHeight" to the amount of pixels heigh that "thisSelector" is equal to
  • Then update the height of the element with the CSS selector that is equal to "parentID" to the height of "childImageHeight"
